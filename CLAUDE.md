# Claude Instructions

This repository contains the MSAccess Version Control System (VCS) Add-in for Microsoft Access.

For comprehensive guidance on working with this codebase, please read:

- **[AGENTS.md](AGENTS.md)** - Main project guide covering architecture, coding standards, and development workflow
- **[Version Control.accda.src/AGENTS.md](Version%20Control.accda.src/AGENTS.md)** - Guide for working with exported Access database source files

These files contain essential information about:
- Project architecture and component structure
- VBA coding standards and naming conventions
- Error handling patterns
- How to add new features or modify existing behavior
- The interface-based component system (`IDbComponent`)
- Working with exported source file formats (UTF-8 BOM encoding, VBA attributes, etc.)
